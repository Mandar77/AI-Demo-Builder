üîë Key Architectural Insights

üìä DynamoDB = Central Nervous System

Every service reads current state, performs work, writes new state. The session record evolves from initialization to completion, tracking the entire journey.

üîÑ Async Processing

Phases 1-2 are synchronous (quick). Phase 3 uses SQS for async processing because video stitching takes 2-5 minutes. Frontend polls status.
